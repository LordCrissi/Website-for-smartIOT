# IEEE SmartIoT 2026 website

A responsive, multi-page redesign of the IEEE SmartIoT 2026 conference website.

## Preview

Open `index.html` directly in a browser. For a local web server, run one of the following from this folder:

```powershell
py -m http.server 8080
```

Then visit `http://localhost:8080`.

## Files

- `index.html` — conference homepage
- `committee.html` through `contact.html` — redesigned conference subpages
- `styles.css` — responsive layout, visual system and animations
- `script.js` — conference countdown, reveal effects and mobile navigation
- `pages.css` — shared subpage components and layouts
- `pages.js` — subpage content, navigation and interactions

All primary navigation now stays inside the redesigned website. Archived conference-history links and individual legacy news items intentionally continue to reference their original records.
